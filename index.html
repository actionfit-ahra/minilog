📊 구글시트 + Apps Script 연결 (권장)

Apps Script가 Singular API 프록시 + 저장소 역할을 동시에 수행합니다. 싱귤러 키는 Apps Script 내부에 저장되어 노출되지 않습니다.

준비됨. 위 버튼을 클릭하세요.
📁 CSV 직접 업로드 (백업)
📖 Apps Script 설정 가이드

1단계. 구글시트 생성 → 확장 프로그램 → Apps Script

2단계. minilog_apps_script.gs 코드 붙여넣기

3단계. Singular API 키를 SINGULAR_API_KEY 변수에 입력

4단계. initializeSheets() 실행 (최초 1회)

5단계. syncSingularData() 실행 → 데이터 불러오기

6단계. 트리거 설정 → 매일 자동 실행

7단계. 배포 → 웹앱 → URL 복사 → 위에 붙여넣기
