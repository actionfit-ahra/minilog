이벤트 분석

이벤트 CSV + 캠페인 캘린더 + Singular D1 Rev를 결합해서 달성률과 예상 비용, ROI를 계산합니다.
1. 이벤트 Raw Data (CSV)
매체에서 받은 이벤트 로그
필수 컬럼: partner, name, device id
대기 중
2. Singular D1 Rev (선택)
기본은 대시보드 데이터를 자동 사용
별도 CSV가 있으면 업로드 가능
대시보드 데이터 자동 사용 중
3. 캠페인 캘린더 (자동)
슬랙에서 파싱된 캠페인 세팅 이력
(최신 단가 기준 자동 매핑)
대시보드 데이터 기준 — 자동 로드됨
사용 방법
1. 이벤트 Raw Data CSV 업로드 (필수)
2. Singular D1 Rev CSV 업로드 (ROI 계산 시 필요)
3. 캘린더는 대시보드에서 자동으로 가져옴 (캠페인 탭 데이터)
4. "분석 실행" 클릭 → 테이블 생성
먼저 "데이터 업로드" 탭에서 CSV를 업로드하고 분석을 실행해주세요.
